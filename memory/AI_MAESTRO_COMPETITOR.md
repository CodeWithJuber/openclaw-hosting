# Memory: AI Maestro (23blocks) - Key Competitor

**Date**: 2026-02-16  
**Source**: https://github.com/23blocks-OS/ai-maestro

## Summary
AI Maestro is an AI Agent Orchestrator with Skills System that allows managing Claude, Codex or any AI Agent from one dashboard.

## Key Features to Remember
- **One Dashboard**: Manage all AI agents in one place
- **Multi-Machine**: Peer mesh network (no central server)
- **Agent Messaging Protocol (AMP)**: Email-like communication between agents
- **Persistent Memory**: 3-layer system (memory, code graph, docs)
- **Code Graph**: Interactive codebase visualization
- **Kanban Board**: Full project management for agents
- **Gateways**: Slack, Discord, Email, WhatsApp integration
- **Agent Identity**: Custom avatars, personality profiles

## Founder Story
- Started with 35 agents across terminals
- Became "human mailman" between agents
- Now runs 80+ agents building real companies
- Built with Next.js, xterm.js, CozoDB, ts-morph, tmux
